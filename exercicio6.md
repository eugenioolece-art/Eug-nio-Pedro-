# Exercício 6: Complexidade Algorítmica

Considere a funçăo `search(Node* head, int value)`.

**Pergunta:**
Se a lista tiver 1.000.000 de elementos e o valor procurado năo existir, quantos nós o programa precisará visitar? Qual é a classificaçăo Big O desta operaçăo?

**Resposta Esperada:**
O programa precisará visitar todos os 1.000.000 de nós. A complexidade é $O(n)$, onde $n$ é o número de elementos na lista.
